Features of this Post Navigator

* SEO Optimized
* CSS3 Transitions
* Unlimited Color Scheme
* Effective Plugin to increase SERP
* No Coding knowledge Needed, Just plug& play.

Requires WordPress 4.9 and PHP 7.0 or higher

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ucat-next-story` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings -> Next Story screen to configure the plugin
